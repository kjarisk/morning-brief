Anthropic launched ten pre-built Claude agent templates for financial services on May 5, 2026, covering the most common high-value workflows in investment banking, private equity, asset management, and wealth management. The agents include a pitch builder, meeting preparer, earnings reviewer, financial model builder, market researcher, KYC screener, valuation reviewer, general ledger reconciler, month-end closer, and statement auditor. Each ships as both a Claude Cowork plugin (for business users) and a Claude Managed Agent template deployable via the /v1/agents API (for developers).

The data connector story is significant: Anthropic partnered with Dun & Bradstreet, Fiscal AI, Financial Modeling Prep, Guidepoint, IBISWorld, SS&C IntraLinks, Third Bridge, and Verisk to give agents live access to financial data without requiring customers to build their own integrations. The agents also work with Microsoft Excel, PowerPoint, and Outlook through a Claude + Microsoft 365 integration announced at the same time.

This accompanies the broader joint venture with Blackstone, Goldman Sachs, and Hellman & Friedman announced the same week, in which Anthropic is targeting PE-owned professional services firms. The financial-services GitHub repo contains reference code and documentation for developers who want to customize or extend the templates.

Read more at [GitHub](https://github.com/anthropics/financial-services).
