Anthropic published the opening keynote of its Code with Claude developer conference, held today in San Francisco. The session features Boris Cherny (Head of Claude Code) and Angela Jiang (Claude API Product Lead) covering the state of agentic coding, live demos of Claude Code workflows, and a walkthrough of the MCP ecosystem for connecting Claude to local and remote data sources. The keynote is followed by a full day of workshops and one-on-one office hours.

This is Anthropic's first formal developer conference, and the keynote is the primary source for any model or product announcements made at the event today. If Jupiter-v1-p ships alongside the conference, the keynote is where it will be introduced. For developers building on Claude, this is the clearest signal of where the platform is heading in the next quarter.
