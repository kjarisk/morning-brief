Anthropic's first developer conference, Code with Claude, opened today in San Francisco with a full day of workshops, live capability demos, and one-on-one office hours with Anthropic engineers. The event is livestreamed and led by Ami Vora (Head of Product), Boris Cherny (Head of Claude Code), and Angela Jiang (Claude API Product Lead). The technical curriculum centres on three areas: agentic coding workflows using Claude Code and Claude Cowork, the MCP ecosystem and how to connect Claude securely to local and remote data sources, and production reliability practices including evaluation pipelines.

Running in parallel, testing catalogue sources report that Anthropic has been red-teaming a new model codenamed Jupiter-v1-p ahead of a potential launch timed to the conference. No official announcement has been made, but the pattern of pre-event red-teaming — similar to what preceded previous Claude releases — suggests a model drop is possible before end of day. London and Tokyo editions of the conference follow on May 19 and June 10 respectively.
