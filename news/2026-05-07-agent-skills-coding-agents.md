agent-skills is a curated repository of production-grade shell-based skill files for AI coding agents, maintained by Addy Osmani (formerly Google Chrome engineering lead). The repo has 31,200 stars and added 800 today. Each skill is a structured prompt file covering a specific engineering workflow — debugging, code review, architecture decisions, testing strategy, documentation — that you can drop directly into Claude Code, Cursor, or any other agent that supports skill/context files.

The practical value is that these skills encode best practices developed across real engineering teams rather than toy examples. A skill for code review, for instance, will prompt the agent to check for security issues, test coverage gaps, API design consistency, and performance concerns in a structured way — the same checklist a senior engineer would run, packaged as a reusable file.

For developers using Claude Code or Cursor on a React/TypeScript/FastAPI stack, this repo is worth bookmarking. The skills are language-agnostic but many examples reflect modern web and Python patterns. Given Osmani's background at Google, the architecture and performance-oriented skills in particular are battle-tested at scale.

Read more at [GitHub](https://github.com/addyosmani/agent-skills).
